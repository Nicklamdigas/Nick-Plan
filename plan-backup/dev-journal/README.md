# Dev Journal — Nicholas

A private log of my learning journey as a Software Engineering student.
No fluff. Just honest daily entries about what I learned, what broke, what I fixed.

---

## Why this exists
I've been relying on AI to write code for me without understanding it.
That ends now. This journal is proof that I'm changing that.

## Format
Each entry answers:
- What did I work on today?
- What did I actually understand (not just copy)?
- What confused me or broke?
- How did I fix it (or what do I still need to figure out)?
